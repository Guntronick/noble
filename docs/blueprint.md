# **App Name**: AI Merch Catalog

## Core Features:

- Home Page: Displays a landing page with company logo, a slogan, and a 'View Catalog' button linking to the product listing.
- Product Listing: Presents products with a preview of name, image carousel, and brief description. Allows users to choose to display 20, 50, or 100 products per page.
- Product Detail Page: Provides a detailed product view with a large image, smaller selectable images, price, color options, quantity selection, and an 'Add to Cart' button with a confirmation modal.
- Related Products: Suggests related products from the same category. The tool intelligently personalizes recommendations based on viewed products.
- Share and Product Code: Offers social media sharing options (X, Facebook, Instagram, WhatsApp, email) and displays the product code.
- Category Page: Shows product categories as square sections with a background image, category name, and a button to navigate to that category's products.
- Contact Form: Includes a contact form (name, email, phone, message) with an 'Send' button that emails the collected information to a pre-configured email address. Must use NextJS email API routes

## Style Guidelines:

- Primary color: #002529 (Dark Teal), for headers and main elements.
- Secondary color: #598392 (Grayish Blue), complements the primary color.
- Accent color 1: #A9D6E5 (Light Blue), provides a soft contrast, maintaining a professional but tech-forward feel, used for secondary buttons and highlights.
- Accent color 2: #ADC178 (Olive Green), for subtle call-to-action buttons.
- Neutral color: #F0EBD8 (Beige), provides a clean background for text readability.
- Background color: #FFFFFF (White), provides a professional backdrop.
- Headline font: 'Poppins' (sans-serif), is used for headers to impart a modern, techy look. Body text: 'Roboto' (sans-serif).
- Simple, clean icons representing product categories and social media platforms.
- Modular design for components like navbar and footer to minimize page reloads and improve performance.
- Subtle transitions and loading animations to enhance user experience.
